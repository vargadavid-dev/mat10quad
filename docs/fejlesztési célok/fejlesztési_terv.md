# Fejlesztési Célok - Matematika 10.

Ez a dokumentum a projekt jövőbeli fejlesztési irányait és tervezett funkcióit tartalmazza.

## 1. Tartalom Bővítése
- [ ] **Magasabb fokú egyenletek:** Harmad- és negyedfokú egyenletek megoldási módszerei.
- [ ] **Szöveges feladatok:** Valós életből vett problémák, amelyek másodfokú egyenletre vezetnek.
- [ ] **Paraméteres egyenletek:** Mélyebb elméleti anyag a paraméteres vizsgálatokhoz.

## 2. Funkcionális Fejlesztések
- [ ] **Felhasználói fiókok:** Haladás mentése adatbázisba (Firebase/Supabase).
- [ ] **többnyelvűség (i18n):** Angol és más nyelvek támogatása.
- [ ] **Sötét mód (Dark Mode):** Rendszerszintű téma követése.

## 3. Gamifikáció és UX
- [ ] **Jelvények (Badges):** Különleges teljesítményekért járó jutalmak (pl. "Hibátlan fejezet", "Gyorsmegoldó").
- [ ] **Animációk finomítása:** Még több interaktív visszajelzés a feladatoknál.
- [ ] **Hanghatások:** Opcionális hangeffektek helyes/helytelen válaszoknál.

## 4. Technikai Adósság
- [ ] **Tesztek írása:** Unit tesztek (Vitest) a komponensekhez és a logikához.
- [ ] **Kódoptimalizálás:** Teljesítmény javítása és refaktorálás.
